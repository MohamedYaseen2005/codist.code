Write a program to the sum of n natural numbers.

Concept:

Here is a 10-line definition of the problem: Write a program to calculate the sum of the first n natural numbers. The program should take an integer input n from the user. The input n should be a positive integer. The program should calculate the sum of all natural numbers from 1 to n. The sum should be calculated using a loop or a formula.
